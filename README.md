## Overview
The **CDI Program** is a laboratory automation tool designed to control and monitor electrochemical systems. The tool was developed for capacitive deionization (CDI)
experiments, but it can be used for any system that requires controlled voltage or current segments and multi-parameter monitoring. The program integrates hardware components
to run user-defined experimental parameters, providing operational control of a programmable power supply, peristaltic pump, and data acquisition devices.

---

## Features
- [Feature 1: Automatic hardware control of power supply, peristaltic pump, and probes]  
- [Feature 2: Multiple experimental workflows with 2 or 4 segments per cycle]  
- [Feature 3: Real-time monitoring and plotting of voltage, current, flow rate, conductivity, pH, and temperature]  
- [Feature 4: Automated cycle control with initialization and post-run stabilization]  
- [Feature 5: Data logging and export to a single synchronized Excel file]  
- [Feature 6: Dual-cell control for simultaneous experiments under different voltage/current conditions]  
- [Feature 7: User-friendly Excel-based interface for modifying experimental inputs without editing code]
